# Entregas Trainee 26.1 - TerraLab

Este repositório concentra todas as entregas práticas e desenvolvimentos realizados durante o programa de trainee.

* **Airflow:** Para orquestração das tarefas ETL.
* *(Mais ferramentas serão adicionadas ao longo das semanas)*
